setTimeout и setInterval — это методы, предоставляемые браузером (и средой Node.js) для планирования выполнения функций асинхронно. 

**setTimeout(callback, delay, ...args):**
  * Выполняет функцию callback **один раз** после указанной задержки delay в миллисекундах.  
  * Любые дополнительные аргументы ...args будут переданы в callback при ее вызове.  
  * Возвращает числовой ID таймера, который можно использовать для отмены выполнения с помощью clearTimeout(timerId).
```js
console.log("Start"); 
 
const timerId \= setTimeout(() \=\> {  
  console.log("Executed after 2 seconds");  
}, 2000);  

// clearTimeout(timerId); // Можно отменить до выполнения  
console.log("End (this logs before setTimeout callback)");
```

**setInterval(callback, delay, ...args):**
  * Выполняет функцию callback **многократно** с указанным интервалом delay в миллисекундах между вызовами.  
  * Любые дополнительные аргументы ...args будут переданы в callback при каждом ее вызове.  
  * Возвращает числовой ID интервала, который можно использовать для остановки повторных выполнений с помощью clearInterval(intervalId).
```js
let count = 0;  
const intervalId = setInterval(() => {  
  count++;  
  console.log('Executed ${count} times');  
  
  if (count >= 3) {  
    clearInterval(intervalId); // Остановка после 3 выполнений  
    console.log("Interval stopped");  
  }  
}, 1000);
```

 **Важно:**
  * Задержка delay не является гарантированной. Это минимальное время ожидания. Если основной поток JavaScript занят, выполнение колбека может быть отложено.  
  * Если callback в setInterval выполняется дольше, чем delay, то следующие вызовы могут начать "накапливаться" или выполняться сразу после завершения предыдущего, без строгого соблюдения интервала. Для более точного контроля над длительными операциями, выполняемыми по интервалу, иногда используют рекурсивный setTimeout:  
```js
function doSomethingRepeatedly() {  
  // ... ваш код ...  
  setTimeout(doSomethingRepeatedly, 1000); // Запланировать следующий вызов  
}  
setTimeout(doSomethingRepeatedly, 1000);
```
