
JavaScript имеет примитивные типы данных и один сложный тип (объект).

   **Примитивные типы (Primitive Types):**

  1. **Number**: Для любых чисел: целочисленных и с плавающей точкой (e.g., 10, 3.14, NaN, Infinity).  
  2. **String**: Для текстовых данных (e.g., "hello", 'world').  
  3. **Boolean**: Логический тип, true или false.  
  4. **Undefined**: Значение, присваиваемое переменной, которая была объявлена, но не инициализирована. Также это значение функции, которая ничего не возвращает явно.  
  5. **Null**: Представляет намеренное отсутствие какого-либо объектного значения. Означает "пусто" или "неизвестно".  
  6. **Symbol (ES6+)**: Уникальный и неизменяемый тип данных, часто используется для создания уникальных идентификаторов свойств объектов.  
  7. **BigInt (ES2020+)**: Для представления и работы с целыми числами произвольной длины, превышающими максимальное безопасное целое число в Number.  

* **Сложный тип (Complex Type):**

  1. **Object**: Используется для хранения коллекций данных (ключ-значение) и более сложных сущностей. Включает в себя массивы (Array), функции (Function), даты (Date), регулярные выражения (RegExp) и другие объекты. Функции в JavaScript также являются объектами.
